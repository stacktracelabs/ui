# Switch

A control that allows the user to toggle between checked and not checked.

<ComponentPreview name="Switch" />

## Installation

```shell
php artisan ui:add switch
```

## Usage

```vue
<template>
  <Switch />
</template>

<script setup lang="ts">
import { Switch } from '@/Components/Switch'
</script>
```
