# Text Area

Displays a form textarea or a component that looks like a textarea.

<ComponentPreview name="Textarea" />

## Installation

```shell
php artisan ui:add textarea
```

## Usage

```vue
<template>
  <Textarea />
</template>

<script setup lang="ts">
import { Textarea } from '@/Components/Textarea'
</script>
```
