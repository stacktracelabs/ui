# Checkbox

A control that allows the user to toggle between checked and not checked.

<ComponentPreview name="Checkbox" />

## Installation

```shell
php artisan ui:add checkbox
```

## Usage

```vue
<template>
  <Checkbox />
</template>

<script setup lang="ts">
import { Checkbox } from '@/Components/Checkbox'
</script>
```
