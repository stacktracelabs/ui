# StackTrace UI

Build Laravel interfaces that feel considered from the controller all the way to the final interaction.

StackTrace UI brings together backend-driven UI primitives, focused Vue utilities, and a growing collection of beautifully composed shadcn-vue components. Use PHP objects to shape tables, navigation, actions, and feedback, then bring them to life in Inertia with components you install directly into your application.

It is designed for teams who want the freedom of application-owned components without rebuilding the same interface patterns for every project—and for AI agents that need clear guidance to use those patterns consistently.

[Explore the documentation](https://ui.stacktrace.sk/docs) or [browse the component registry](https://ui.stacktrace.sk/r/registry.json).
